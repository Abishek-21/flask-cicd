CI/CD Pipeline Demo v1.0.0

About

This app is automatically tested, built, and deployed using GitHub Actions CI/CD pipeline. Every push to the main branch triggers the full pipeline.

Pipeline Steps

1. Push code to GitHub

2. GitHub Actions runs automated tests

3. Docker image is built automatically

4. Image is pushed to Docker Hub

API Endpoints

GET /api/status - App status
GET /api/health - Health check
GET /api/info - Project info

Tech Stack

Python Flask Docker GitHub Actions Docker Hub AWS EC2
